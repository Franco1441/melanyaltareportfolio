# Melany Altare Portfolio

Portfolio web para una cliente de diseno multimedial, desarrollado como sitio one-page con foco en identidad visual, transiciones y presentacion de trabajos.

## Demo
- Produccion: https://melanyportfolio.netlify.app
- Repo: https://github.com/Franco1441/melanyaltareportfolio

## Stack
- HTML5
- CSS3 (animaciones y layout responsive)
- JavaScript vanilla
- Google Fonts

## Funcionalidades
- Hero visual con composicion tipografica personalizada
- Navegacion one-page (`Sobre mi`, `Trabajos`, `Contacto`)
- Seccion de proyectos destacados con imagenes
- Seccion de contacto con accesos directos (mail, Instagram, WhatsApp)
- Interacciones de cursor y reveal on scroll

## Estructura
El codigo principal del sitio esta en:

- `Portfolio Mela/index.html`
- `Portfolio Mela/styles.css`
- `Portfolio Mela/*.png|*.jpeg` (assets visuales)

## Ejecutar en local
1. Clonar repo:
   ```bash
   git clone https://github.com/Franco1441/melanyaltareportfolio.git
   cd melanyaltareportfolio
   ```
2. Levantar servidor estatico (opcional recomendado):
   ```bash
   npx serve "Portfolio Mela"
   ```
3. Abrir en navegador.
